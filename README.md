# i3lock-nordic

__A [Nordic](https://github.com/arcticicestudio/nord) Theme for [i3lock-color](https://github.com/Raymo111/i3lock-color)__

![](https://raw.githubusercontent.com/mipy1/i3lock-nordic/main/Screenshot.png)

## Installation:

Clone this repo and add a shortcut to it in your i3 config file (or whatever you want to do with it tbh)
e.g. (from [my i3 config](https://github.com/mipy1/dotfiles/tree/main/i3)):
```
bindsym $mod+p exec "'/home/mipy/.config/i3lock/i3lock-colour-nordic.sh' && systemctl suspend"
```

### Dependencies:

Depends on [i3lock-color](https://github.com/Raymo111/i3lock-color)
