# i3lock-nordic

__A Nordic Theme for [i3lock-color](https://github.com/Raymo111/i3lock-color)__

![](https://raw.githubusercontent.com/mipy1/i3lock-nordic/main/Screenshot.png)

## Installation:

Clone this repo and add a shortcut to it in your i3 config file (or whatever you want to do with it tbh)

### Dependencies:

Depends on [i3lock-color](https://github.com/Raymo111/i3lock-color)
